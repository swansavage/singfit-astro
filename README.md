# SingFit STUDIO Caregiver Demo

Static landing page prototype.

## Structure

- `index.html` — page markup
- `css/styles.css` — all page styles
- `js/main.js` — carousel/menu behavior
- `assets/` — extracted images, logos, badges, and device mockups

## Local preview

Open `index.html` directly in a browser, or run a small static server:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Quick deploy

Drag this whole folder into Netlify Drop, or zip and upload the folder contents.
